Light Survey へ ようこそ

心地よく働くための月次コンディションサーベイ。 毎月10分で、自分のコンディションと（メンバーは）リーダー評価を回答するだけ。 早期に気づき、早期にフォローできる組織を一緒につくります。

月次サイクル 10分以内で完了 全社員対象
1

このサーベイの目的

なぜ毎月、回答するのか
📊

コンディションを継続的に把握する

一人ひとりが心地よく働けているか。月に一度「最近どう？」を構造化して聞く仕組みです。

🎯

マネジメントを見える化する

メンバーからの評価と役員の期待を組み合わせて、リーダーのマネジメントが機能しているか測定します。

🤝

早期に気づき、早期にフォロー

コンディションが低下した時に、1on1等で素早く対応できるようにします。

2

3つのロール

自分のロールは管理者から事前に案内されます
ロール 対象者 サーベイで答えること
メンバー 現場メンバー リーダー評価 ＋ 自身のコンディション
リーダー チームを持つ管理職 自己評価 ＋ チーム観察 ＋ 自身のコンディション
役員 経営層 / 執行役員 各リーダーへの期待比重（4軸の配分）
3

利用ステップ

ログインから回答完了まで
STEP1

本番 URL を開く

ブラウザで以下の URL にアクセスしてください。

https://light-survey.light-5d5.workers.dev/

ブックマーク登録しておくと次回からアクセスがスムーズです

STEP2

Google アカウントでサインイン

「Google でログイン」ボタンを押すと Google のアカウント選択画面に進みます。@light-inc.com の会社アカウントを選択してログインしてください。

個人 Gmail ではログインできません。必ず会社の Google アカウントを使用してください

STEP3

ホーム画面に到達

ログインに成功すると、自分のロールに応じたホーム画面が表示されます。左上に自分の名前と所属チームが見えればOKです。

ロール表示が違う場合は管理者（Slack: #light-survey）までご連絡ください

STEP4

サーベイページを開く

左メニューの「サーベイ」をクリック。当月のサーベイカードが表示されます。

サーベイカードに「未回答」が表示されている期間内のものを選んでください

STEP5

設問に答える（選択肢を押すだけ）

チャット形式で設問が表示されます。しっくりくる選択肢を1つ押してください。次の設問に自動で進みます。

各設問は4段階（とても感じている / 概ね / 少し / 感じていない 等）

STEP6

自由記述（任意）

最後にリーダーへ伝えたいことがあれば入力欄に記入してください。特にない場合は「特になし」ボタンを押せばOKです。

自由記述はリーダー本人と一部の権限者のみが閲覧します

STEP7

送信して完了

全設問に回答し終えると完了画面に移ります。「回答を送信して終了」ボタンを押すと、回答が保存されます。

送信後も期間内なら何度でも修正可能です

4

何を答えるのか

ロールごとに設問の内容が変わります
ロール 答える内容
メンバー
  1. 自分のリーダーへの評価（目標達成への推進 / 強みの活用 / メンバーの育成 / 他チーム・経営層との連携）
  2. 自身のコンディション（帰属意識 / やりがい / 風土 / 成果実感）
リーダー
  1. 自分のマネジメント自己評価
  2. チーム全体の観察
  3. 自身のコンディション（帰属意識 / やりがい / 風土 / 成果実感）
役員 各リーダーへの期待を4つの観点（目標達成への推進 / 強みの活用 / メンバーの育成 / 他チーム連携）で合計100%で配分
5

困った時は

気軽にお声がけください

🔐 ログインできない / ロールが違う

管理担当（HR）に Slack（個別チャンネル等）でご連絡ください。

❓ 設問の意味が分かりづらい

「迷いがある」「物足りない」など、しっくりくる選択肢を選んでOKです。深く考えすぎず、率直に。

率直な回答が、組織をよくします
